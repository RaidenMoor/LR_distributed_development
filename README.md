# Репозиторий с лабораторными работами по дисциплине "Основы распределенной разработки"
## Лабораторная работа №1: Разработка клиент-серверного приложения с применением сокетов на java
1. Реализована клиентская часть приложения в методе main() отдельного класса. Приложение-клиент устанавливает через сокет соединение с приложением-сервером, после чего циклически передает серверному приложению координаты n-мерного вектора и ждет от него результат вычисления длины этого вектора. После получения ответа приложение-клиент переходит к новому витку цикла передачи. 

2. Формат передачи через сериализацию

3. Приложение-клиент передает несколько векторов, организован цикл передачи векторов, в каждой итерации передается один вектор.

4. Признак окончания взаимодействия клиента с сервером - передача флага

5. Реализована (в отдельном модуле компиляции) серверная часть приложения в рамках модели последовательной обработки запросов.

6. Логика функционирования серверного приложения следующая: серверное приложение принимает запрос от приложения-клиента, получает от него координаты n-мерного вектора, вычисляет длину вектора, отправляет результат приложению-клиенту, затем переходит к ожиданию координат следующего вектора в n-мерном пространстве от того же клиента. Если длина вектора равна 0, то серверное приложение отправляет клиенту сообщение об ошибке.

7. Реализована (в отдельном модуле компиляции) серверная часть приложения в рамках модели параллельной обработки запросов. Логика функционирования серверного приложения аналогична последовательному.
